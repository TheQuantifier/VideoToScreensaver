# VideoToScreensaver v1.1

## Windows Installer
- Download `VideoToScreensaver-Setup.exe` from this release.
- Run the installer and launch `VideoToScreensaver` from Start Menu.

## Updates
- Updated post-install instructions in the app to tell users to confirm `"<yourfilename>_vts.scr"` is selected in Screen Saver Settings (instead of `VideoToScreensaver`).

## Notes
- Windows only.
- Unsigned build: Windows SmartScreen may show a warning.
